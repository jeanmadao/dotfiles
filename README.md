# Dotfiles
My dotfiles for my personal Archlinux Hyprland setup.

## Dependencies
All of the following packages are used for this config
- [hyprland](https://archlinux.org/packages/extra/x86_64/hyprland/)
- [hyprpaper](https://archlinux.org/packages/extra/x86_64/hyprpaper/)
- [kitty](https://archlinux.org/packages/extra/x86_64/kitty/)
- [zsh](https://archlinux.org/packages/extra/x86_64/zsh/)
- [polkit-kde-agent](https://archlinux.org/packages/extra/x86_64/polkit-kde-agent/)
- [dunst](https://archlinux.org/packages/extra/x86_64/dunst/)
- [geoclue](https://archlinux.org/packages/extra/x86_64/geoclue/)
- [gammastep](https://archlinux.org/packages/extra/x86_64/gammastep/)
- [dolphin](https://archlinux.org/packages/extra/x86_64/dolphin/)
- [wofi](https://archlinux.org/packages/extra/x86_64/wofi/)
- [grim](https://archlinux.org/packages/extra/x86_64/grim/)
- [slurp](https://archlinux.org/packages/extra/x86_64/slurp/)
